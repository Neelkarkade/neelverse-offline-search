# Neelverse Offline Q&A Search System

A full‑stack project built with **Next.js + TypeScript** that demonstrates offline Q&A search using a local JSON dataset.  
Designed for interview demos to showcase API routes, environment setup, and Postman testing.

---

## 🚀 Features
- Offline search API (`/api/ask`) that reads from `data/search.json`
- Partial keyword matching for flexible queries
- Tested with Postman for reproducible demo flows
- Clean project structure with modular API routes
- Git + GitHub version control for professional workflow
